# Number Guessing Game

## Description
This is a simple python project where
the computer generates a random number between 1 to 10.
The user tries to guess the number.

##  how it works 
1. The program generates a random number
2. the user enter a guess
3. Program tells if the guess is too high,
too low , or correct.

## Concepts used 
- Python
- random Module
- if-else conditions
- loops
